# Glossary

- Texture: A 2D rectangle of pixels. In HTML, a `<canvas>` element represents a texture on the page. [source](https://webgpufundamentals.org/webgpu/lessons/webgpu-fundamentals.html#a-drawing-triangles-to-textures)
- Shader: A function that is executed on the GPU. [source](https://webgpufundamentals.org/webgpu/lessons/webgpu-fundamentals.html#a-drawing-triangles-to-textures)
- Vertex shader: A shader that computes positions of vertices, which are used to draw points, lines or triangles. [source](https://webgpufundamentals.org/webgpu/lessons/webgpu-fundamentals.html#a-drawing-triangles-to-textures)
- Fragment shader: A shader that computes colors for pixels of a points, lines or triangles. [source](https://webgpufundamentals.org/webgpu/lessons/webgpu-fundamentals.html#a-drawing-triangles-to-textures)
- Compute shader: A shader that is repeatedly executed for some iteration. [source](https://webgpufundamentals.org/webgpu/lessons/webgpu-fundamentals.html#a-draw-diagram)
- Pipeline: An internal GPU pipeline containing the vertex shader and/or fragment shader. [source](https://webgpufundamentals.org/webgpu/lessons/webgpu-fundamentals.html#a-draw-diagram)
- Bind group: A handle for shaders to access resources indirectly. [source](https://webgpufundamentals.org/webgpu/lessons/webgpu-fundamentals.html#a-draw-diagram)
